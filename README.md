# Part 107 Flashcards

Study tool for the FAA Part 107 Remote Pilot Knowledge Test — 400+ flashcards covering all testable topics, with a built-in practice exam mode.

**Live site:** https://kevin-pan-skydio.github.io/part-107-flash-cards/

## Features

- **412 flashcards** organized into 5 categories:
  - **CFR (§)** — 14 CFR Part 107 regulations, certificates, waivers & compliance
  - **Airspace (◎)** — Airspace classifications, TFRs, NOTAMs & flight restrictions
  - **Sectional Charts** — Chart reading, symbology, navigation & map interpretation
  - **Airport** — Airport operations, traffic patterns & communication
  - **Weather** — Weather theory, reports, forecasts & environmental effects
- **Practice exam mode** — Timed mock exams with 10, 25, 50, 60, or 72 questions
- **Keyboard navigation** — Space to flip, arrow keys to navigate, S to shuffle, R for random
- **Expandable context** — Many cards include additional context for deeper understanding
- **Quick-reference panels** — Weather notation and airspace tables accessible while studying
- **Sectional chart figures** — SVG visual cards for airport surface markings

## Usage

Open `index.html` in a browser. No build step or dependencies required — it's a single self-contained HTML file.

## Deployment

Hosted on GitHub Pages from the `main` branch. Push to `main` to deploy.
